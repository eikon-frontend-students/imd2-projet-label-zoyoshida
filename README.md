# imd2-projet-label

## Installation
1. Cloner le repository
2. Ouvrir le dossier avec Visual Studio Code 
3. Ouvrir une fenêtre de terminal dans Visual Studio Code et entrer la commande suivante :
```bash
starterkit-create
```
Plus d'informations à propos du terminal et du starterkit:
- [Terminal](https://www.notion.so/eikon-imd/Terminal-d4858d6e49514c868726bad9566cf85f?source=copy_link)
- [Starterkit](https://www.notion.so/eikon-imd/Starterkit-710621ff272b47b1a604e9302077f614)
